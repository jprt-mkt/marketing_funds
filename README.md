# Marketing Fund App

Aplicação web para gestão de fundo de marketing, construída com React + TypeScript + Vite.

## 🛠️ Stack Tecnológica

- **Framework:** React 18 + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS + Shadcn UI
- **Routing:** React Router DOM
- **Data Fetching:** TanStack Query (React Query)
- **Icons:** Lucide React
- **Data Viz:** D3.js
- **Package Manager:** Bun

## 🚀 Como rodar localmente

### Pré-requisitos
- Node.js 18+ ou Bun
- Git

### Instalação

\`\`\`bash
# Clonar o repositório
git clone https://github.com/SEU-USUARIO/marketing-fund-app.git
cd marketing-fund-app

# Instalar dependências
bun install
# ou
npm install

# Rodar em modo desenvolvimento
bun run dev
# ou
npm run dev
\`\`\`

O app estará disponível em `http://localhost:5173`

## 📜 Scripts Disponíveis

| Comando | Descrição |
|---------|-----------|
| `bun run dev` | Inicia servidor de desenvolvimento |
| `bun run build` | Gera build de produção |
| `bun run preview` | Preview do build de produção |
| `bun run lint` | Executa linter |
